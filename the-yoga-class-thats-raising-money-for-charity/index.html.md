# The Yoga Class That's Raising Money for Charity ...

According to [Stylist,](https://www.stylist.co.uk/beauty/cuddle-core-bunny-yoga-is-the-latest-fitness-trend-to-go-viral/137309) there's a new yoga class in Richmond that involves bunnies! The class contains twenty-seven women and ten adorable little rabbits.

Why does such a class exist? Well, i's meant to help those bunnies find homes, since the animals are overpopulated. Of course, it also helps some of the women relax, since that's the point of yoga.

So far, the yoga classes have helped raise over $1,000 for Bandaids for Bunnies. If you're going to work out, you might as well work out for a good cause!
